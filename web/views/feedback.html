Feedback is a "go on rails" framework that has routes, controllers, views, layouts, and a non-ORM database query/update system.

It was started on Feb 2, 2020 and is being actively maintained. The latest commit was Jul 5, 2023.

To define routes you first pick a top level name:

r.Paths["/"] = app.HandleWelcome
r.Paths["aliens"] = app.HandleAliens
r.Paths["species"] = app.HandleSpecies
r.Paths["planets"] = app.HandlePlanets

Every POST, PUT, GET, DELETE, PATCH route for /aliens will be in app.HandleAliens. And you get a max of two levels of /aliens/:second/:third

If you need /aliens/:second/:third/:fourth rethink what you are doing a make a new top level name and break it up better.
